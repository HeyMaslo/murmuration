# MurmurationJS

A JavaScript mush pot and library to murmurate things.

## Murmuration

Definition: 
murmuration (noun)
məːməːˈreɪʃ(ə)n

1. the action of murmuring.
"the murmuration of a flock of warblers"

2. a flock of starlings. (rare)
